# chat-interface

Chat UI built in React. No backend — messages stay in memory.

Built as a UI exercise: message bubbles, timestamps, typing indicator, reactions, file attachment UI (no actual upload).

## Features

- Chat bubbles with sender/receiver distinction
- Timestamps
- Typing indicator animation
- Emoji reactions on hover
- Simulated bot replies (echoes your message after a delay)
- Dark/light mode

## Run

```bash
npm install && npm run dev
```
